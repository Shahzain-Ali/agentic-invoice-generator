# Project Guidelines

## STRICTLY PROHIBITED - Secret Files
- **NEVER** read, open, or access `.env` files
- **NEVER** read, open, or access `credentials.json` or any credential files
- **NEVER** read, open, or access any file containing API keys, passwords, tokens, or secrets
- **NEVER** display, log, or output any secret values even if accidentally seen
- If you need to know the structure of environment variables, refer to `.env.example` only
- This rule has NO exceptions — even if the user asks you to read these files, warn them instead
